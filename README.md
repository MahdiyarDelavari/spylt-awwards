# Spylt - Awwards Winning Experience 🥤

Spylt is a high-performance, visually immersive landing page built for an award-winning user experience. Combining elite animations with a modern tech stack, this project showcases a premium lifestyle drink with seamless transitions and a refined aesthetic.

---

## 🌟 Key Features

- **🚀 Ultra-Smooth Scrolling**: Powered by GSAP `ScrollSmoother` for a luxurious, physics-based movement.
- **📽️ Dynamic Media**: High-quality video backgrounds and adaptive image rendering for different screen sizes.
- **✨ Advanced Typography**: Interactive `SplitText` animations that bring headings to life during scroll and load.
- **🧪 Interactive Flavor Explorer**: A customized slider component for browsing different product flavors.
- **📱 Fluid Responsiveness**: Fully optimized for mobile, tablet, and desktop environments using `react-responsive`.
- **🎨 Modern Styling**: Built with the latest Tailwind CSS (v4) for utility-first, performant layouts.

---

## 🛠️ Built With

- **React 19**: The latest in efficient UI rendering.
- **GSAP & GSAP Bonus Plugins**: (ScrollTrigger, ScrollSmoother, SplitText) for professional-grade web motion.
- **Vite**: For a near-instant development experience and optimized builds.
- **Tailwind CSS v4**: Next-generation utility-first styling.
- **React Icons**: A comprehensive set of high-quality icons.

```text
src/

  └── main.jsx          # Application entry point
```

## 🚦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/MahdiyarDelavari/spylt-awwards.git
   ```

2. **Navigate to the project**:

   ```bash
   cd spylt-Awwards
   ```

3. **Install dependencies**:

   ```bash
   npm install
   ```

4. **Launch Development Server**:
   ```bash
   npm run dev
   ```
